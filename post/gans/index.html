Comparing GAN-Hacks

Thu, Mar 8, 2018 Deep Learning, GAN, Machine Learning

I finally implemented a vanilla Generative Adversarial Network. GANs are notoriously hard to train, and I decided to find out how different tricks worked.

Dataset used was MNIST. The architecture was pretty simple: a few fully connected layers, as the data is not very complex. Both the discriminator and generator had similar number of parameters.
